Hotel Management Backend (Node.js + Express + MongoDB)

A fully functional Hotel Management Backend API built using Node.js, Express.js, and MongoDB.
This backend system helps manage rooms, bookings, customers, staff, and authentication with secure workflows using JWT tokens and bcrypt password hashing.
